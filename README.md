# 微信或支付宝小程序中将less编译成acss
``` This project is fork from https://github.com/echo008/wxss-cli and changed by Hranharry, Thinks to Origin Author Echo```
## 安装

```console
$ npm install -g mini-less-cli
```
## 使用

> demo 为开发目录
```
-a 选项生成支付宝acss
$ mini-less -a ./demo
```
```
-w 选项生成微信wxss
$ mini-less -w ./demo
```
> 项目会自动监听less文件变更并生成对应样式文件
